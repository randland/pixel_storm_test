# Lesson 02-03: Lighting Basics

> **Section**: Scene Fundamentals
> **Status**: `[ ]` Planned
> **Estimated Time**: 1-2 hours

## Learning Objectives

- [ ] Understand different light types and when to use each
- [ ] Know how lights interact with materials
- [ ] Use light helpers for debugging
- [ ] Create a `useLighting` composable

## Prerequisites

- Lesson 02-02: Cameras & Controls

## Target Artifacts

| Artifact | Purpose |
|----------|---------|
| `useLighting` | Composable for light setup |
| Light presets | Common lighting configurations (studio, outdoor, etc.) |

## Key Concepts

*To be developed during lesson*

- AmbientLight (global fill)
- DirectionalLight (sun-like, parallel rays)
- PointLight (bulb-like, radiates from point)
- SpotLight (cone of light)
- Light helpers for visualization
